# Jójárt Sándor villanyszerelő – weboldal

Modern, kék fókuszú bemutatkozó oldal Vite + React segítségével. Tartalmaz navbar-t, hős szekciót, szolgáltatásokat, bemutatkozást, visszajelzéseket és kapcsolatfelvételt.

## Fejlesztői indítás

Telepítés:

	npm install

Fejlesztői szerver indítása:

	npm run dev

Build:

	npm run build

Előnézet:

	npm run preview
